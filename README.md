## 🛠️ Tech Stack

- **Frontend**: React (Vite) + TypeScript  
- **Styling**: Tailwind CSS + shadcn/ui  
- **Animations**: Framer Motion  
- **Email Service**: EmailJS  
- **Deployment**: Vercel
